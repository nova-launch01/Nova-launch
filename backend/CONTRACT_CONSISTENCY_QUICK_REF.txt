------------
npm test contract-consistency                    # Run all tests
npm test contract-consistency -- --coverage      # With coverage
npm test contract-consistency -- --watch         # Watch mode
npm test contract-consistency -- --verbose       # Verbose output

VALIDATION RULES
----------------
Token Name:    1-32 characters
Token Symbol:  1-12 characters
Decimals:      0-18
Amount:        > 0
Batch Size:    1-100
Address:       G + 55 chars

NUMERIC PRECISION
-----------------
Stellar: 7 decimals (stroops)
1 XLM = 10,000,000 stroops
Always use BigInt for amounts
Convert to string for API

TIMESTAMPS
----------
Contract: Unix seconds
Backend:  Date object
API:      ISO 8601 string

Conversions:
Contract → Backend: new Date(ts * 1000)
Backend → Contract: Math.floor(date.getTime() / 1000)
Backend → API:      date.toISOString()

---------------
